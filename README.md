# 📊 Pipeline de Análise de Dados de Vendas | Python, SQL e Power BI

Este projeto apresenta um fluxo completo de análise de dados, desde o tratamento de dados brutos até a construção de um dashboard interativo para apoio à tomada de decisão.

---

## 📌 Visão Geral

O objetivo deste projeto foi transformar dados brutos de vendas em informações relevantes para análise de desempenho e geração de insights.

Para isso, foram utilizadas diferentes ferramentas ao longo do processo:

* **Python (Pandas)** para limpeza e transformação dos dados;
* **SQLite + SQL (via DBeaver)** para armazenamento e consultas analíticas;
* **Power BI** para criação de visualizações e indicadores estratégicos.

---

## 🎯 Objetivos do Projeto

* Limpar e padronizar dados de vendas;
* Estruturar os dados para análise em banco de dados relacional;
* Aplicar consultas SQL para extração de métricas de negócio;
* Desenvolver um dashboard interativo para análise de desempenho de vendas.

## 🔄 Fluxo do Projeto

### 1️⃣ Tratamento e Limpeza dos Dados (Python + Pandas)

Nesta etapa foram realizadas atividades de preparação dos dados para análise:

* Remoção de registros duplicados;
* Tratamento de valores nulos;
* Padronização de colunas;
* Conversão de tipos de dados;
* Organização dos dados para importação no banco.

---

### 2️⃣ Banco de Dados e Consultas Analíticas (SQLite + SQL)

Após a limpeza dos dados:

* Importação dos dados para o banco SQLite;
* Criação de consultas SQL para análise de vendas;
* Utilização de funções como:

  * `SUM()`
  * `GROUP BY`
  * `ORDER BY`
  * Filtros condicionais (`WHERE`);
* Validação da consistência dos dados antes da etapa de visualização.

---

### 3️⃣ Dashboard Interativo (Power BI)

Com os dados tratados e validados, foi desenvolvido um dashboard para análise de vendas contendo:

#### KPIs Estratégicos

* 💰 Faturamento Total
* 📦 Total de Vendas
* 🌎 Quantidade de Países
* 🚗 Linhas de Produto

#### Visualizações

* Evolução do faturamento ao longo do tempo;
* Receita por linha de produto;
* Distribuição de vendas por país;
* Filtros interativos por:

  * País
  * Produto
  * Mês
